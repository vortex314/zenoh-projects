# USe OTA ( Over the Air updates ) via Zenoh
## Features
- Send new Image to ESP32 ( or other )
- Send in 1 Publish message using PSRAM on ESP32 CAM
- works together with OtaActor ( zenoh-esp32 ) sub-project
